# NIST CSF 2.0
_Stands for **N**ational **I**nstitute of **S**tandards and **T**echnology **C**yber**s**ecurity **F**ramework._

_This work is done cited from NIST CSF website which is available online at [www.nist.gov/cyberframework](https://www.nist.gov/cyberframework)._

### CSF Core
_It `organizes cybersecurity values at highest level`, meaning mentions the core values in most undestandable language or term._
